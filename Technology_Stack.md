# Technology Stack
## Flask
Used to build the backend web application and handle routing.
## SQLite
Stores users, complaints, and system data.
## TF-IDF
Converts complaint text into numerical features for machine learning.
## Scikit-learn
Used to train and use machine learning models.
## HTML
Creates webpage structure.
## CSS
Adds styling to webpages.
## Bootstrap
Makes the interface responsive and attractive.
## Git
Tracks project changes.
## GitHub
Stores project online and supports collaboration.
